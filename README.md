# Number-Base-Converter
A simple Python terminal application for converting numbers between decimal, binary, hexadecimal, and octal.

This project was created as part of my programming learning journey. The goal was to practice functions, loops, error handling, and user interaction through the terminal.

## Features

The program can convert numbers between different bases:

- Decimal → Binary
- Decimal → Hexadecimal
- Decimal → Octal
- Binary → Decimal
- Hexadecimal → Decimal
- Octal → Decimal

It also includes:

- Interactive menu
- Error handling for invalid inputs
- Colored terminal interface using the Rich library

## Technologies Used

- Python
- Rich (for colored terminal output)

## How to Run

1. Clone this repository: git clone https://github.com/brunorafa-2/number-base-converter.git

2. Install the required library: pip install rich

3. Run the program: python main.py
